Notes from tutorial

Compiling

Direct compile: "tsc script.ts script.js"

Direct compile short: "tsc script.ts"

Watch for change to compile: "tsc script.ts -w"

Watch for change to compile: "tsc script.ts -w"

Watch for change all files to compile: "tsc -w"

Workflow for public/src folders

Put files to public/source folders

Add init file with command: "tsc --init"

Allow rootDir and public dir and specify folder names

  1. "outDir": "./public"
  2. "rootDir": "./source"

Option to only rootDir files can go inside outDir folder: add ' "include": ["source"] ' to end of tsconfig.json file.

Option to leave out comments: remove comments on ' "removeComments": true, '.

Modular JS

To allow modules in JS we need to edit the script link: " < script type="module" src='app.js' > "

Also we need to edit the tsconfig.json file: "target": "es6", and "module": "es2015",
